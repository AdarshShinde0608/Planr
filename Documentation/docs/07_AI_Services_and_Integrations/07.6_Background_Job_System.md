# Purpose

The Background Job System executes all asynchronous and long-running operations that should not block the user interface.

Instead of executing heavy AI operations inside API requests, they are converted into jobs and processed by dedicated workers.

This enables:

* Fast API responses
* Reliable AI processing
* Automatic retries
* Horizontal scalability
* Continuous AI learning
* Better fault tolerance

---

# Objectives

The Background Job System should:

* Execute asynchronous jobs
* Support delayed execution
* Support recurring jobs
* Process AI workloads
* Retry failed jobs
* Prioritize urgent jobs
* Scale independently
* Monitor worker health

# Best Practices

* Never execute long AI tasks inside HTTP requests.
* Separate queues by responsibility.
* Make every job idempotent.
* Use retry with exponential backoff.
* Monitor queue health continuously.
* Archive completed jobs periodically.
* Use Dead Letter Queues for failed jobs.
* Scale workers independently.
* Keep workers stateless.
* Log every critical job.

---

# Antigravity Development Instructions

When implementing the Background Job System:

1. Use **BullMQ + Redis** as the primary job queue infrastructure.
2. Create dedicated queues for AI, Notifications, Calendar, Analytics, Learning, and Voice services.
3. Build independent worker processes for each queue to allow horizontal scaling.
4. Implement delayed jobs, recurring jobs (Cron), and priority-based scheduling.
5. Ensure all jobs are idempotent and include retry logic with exponential backoff.
6. Implement a Dead Letter Queue (DLQ) for permanently failed jobs and provide tools for inspection and replay.
7. Record metadata for every job, including execution time, retries, and worker assignment.
8. Expose monitoring dashboards (Bull Board, Prometheus, Grafana) for queue health and worker performance.
9. Prevent long-running AI operations from blocking API responses by offloading them to background workers.
10. Design the job system to support future microservice extraction without architectural changes.
