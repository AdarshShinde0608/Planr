# 05.11 UI Screens & Wireframes

## Overview

This document defines the visual architecture of every screen within **The Last-Minute Life Saver**.

It acts as the implementation blueprint for frontend developers, UI designers, and QA engineers.

Every screen must comply with:

- Design System (05.2)
- Routing & Navigation (05.3)
- State Management (05.4)
- UI Component Architecture (05.5)
- Dashboard Architecture (05.6)
- Responsive Design (05.7)
- Animation & Microinteractions (05.8)
- Performance Optimization (05.9)
- Theme System (05.10)

---

# Global Screen Guidelines

Every screen must include:

- Floating Glass Navbar
- Responsive Layout
- Consistent Padding
- Theme Support
- Loading State
- Empty State

- Smooth Page Transition
