# 05.10 Theme System (Light / Dark + Glassmorphism Engine)

## Overview

The Last-Minute Life Saver uses a **dynamic theme system** supporting:

- 🌙 Dark Mode (default, primary experience)
- ☀ Light Mode (clean accessibility mode)
- 🪟 Glassmorphism adaptation in both modes
- 🔁 Real-time theme switching without reload

---

## Design Principle

The UI is NOT static.

Instead:

> Every UI element is driven by **CSS design tokens (variables)** that adapt to theme context.

---

## Theme Modes

### 🌙 Dark Theme (Primary)

Used for:

- Default user experience
- Productivity mode
- AI dashboards

Characteristics:

- Deep navy / charcoal background
- Strong glass blur layers
- Soft neon accents (indigo, cyan)
- High contrast text
- Premium SaaS aesthetic

---

### ☀ Light Theme (Alternative)

Used for:

- Daytime usage
- Accessibility preference
- Print-friendly UI

Characteristics:

- White / soft gray background
- Subtle glass effects
- Reduced blur intensity
- Dark text dominance
- Minimal shadows

---

## Theme Toggle System

### Requirements

- Toggle switch in navbar
- Instant theme switching (no reload)
- Persist user preference
- Respect system theme by default

---

## Theme Detection Logic

Priority order:

1. User saved preference (localStorage)
2. System preference (prefers-color-scheme)
3. Default → Dark mode

---

## CSS Variable Architecture

All colors are defined via tokens:

```css
:root {
  --bg-primary: ...
  --bg-secondary: ...
  --text-primary: ...
  --text-secondary: ...
  --glass-bg: ...
  --glass-border: ...
  --accent-primary: ...
}

Dark Theme Tokens
[data-theme="dark"] {
  --bg-primary: #0B1220;
  --bg-secondary: #111827;

  --text-primary: #F9FAFB;
  --text-secondary: #9CA3AF;

  --glass-bg: rgba(255, 255, 255, 0.08);
  --glass-border: rgba(255, 255, 255, 0.15);

  --accent-primary: #6366F1;
  --accent-secondary: #22D3EE;
}

Light Theme Tokens
[data-theme="light"] {
  --bg-primary: #F9FAFB;
  --bg-secondary: #F3F4F6;

  --text-primary: #111827;
  --text-secondary: #4B5563;

  --glass-bg: rgba(255, 255, 255, 0.55);
  --glass-border: rgba(0, 0, 0, 0.08);

  --accent-primary: #4F46E5;
  --accent-secondary: #0EA5E9;
}
```

## Glassmorphism Rules

### Glass effect MUST adapt to theme:

Dark Mode Glass
Strong blur (16–24px)
Low opacity
Bright border highlight
Light Mode Glass
Medium blur (8–16px)
Higher opacity
Subtle shadow instead of glow
Component Behavior Rules

### All components must:

Use CSS variables only
Never hardcode colors
Never hardcode shadows
Never hardcode opacity

Example rule:

❌ Bad:

background: #111;

✅ Good:

background: var(--glass-bg);
Theme Switching Implementation
Trigger
Navbar toggle button
Keyboard shortcut (optional future)
State Flow
User Click
   ↓
Update Theme State
   ↓
Update data-theme attribute on <html>
   ↓

   ↓
UI re-renders instantly
